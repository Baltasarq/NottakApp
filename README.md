# NottakApp

Note-taking application in plain text files.

This is a Java desktop application.

## Build

Just use Maven.

    $ mvn package
    $ java -jar target/nottakapp-1.0-with-dependencies.jar

## Execution

Build the project with Maven, or download the `.jar` file, and then execute it with `java -jar`.

    $ java -jar NottakApp-1.1.jar 

## Remarks

**Nottakapp** is designed to handle its notes in a shared folder (Dropbox, Drive...), so changes can be carried out in any computer with access to that folder.
